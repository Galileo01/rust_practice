# rust_practice

Rust practice projects.

## Projects

### expr-eval

An arithmetic expression evaluator supporting basic operations (+, -, *, /), exponentiation (^), and parentheses.

```bash
cd expr-eval && cargo run
```

### bestls

A `ls`-like CLI tool that lists directory contents as a styled table or JSON output.

```bash
cd bestls && cargo run [path]
cd bestls && cargo run -- --json [path]
```
